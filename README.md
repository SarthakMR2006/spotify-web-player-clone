# Spotify Web Player UI Clone

This is a UI-only clone of the Spotify Web Player built using HTML, CSS, and JavaScript.

## Live Demo

The live demo is available through GitHub Pages.

## Features

- Spotify-style sidebar
- Header with search bar
- Playlist and album cards
- Bottom music player UI
- Responsive layout
- Basic play/track interaction using JavaScript

## Tech Stack

- HTML
- CSS
- JavaScript

## Note

This project is for educational purposes only. It is a front-end UI clone and does not include real Spotify login, music streaming, or backend functionality.
